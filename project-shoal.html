Shoal

An exploration of human impacts on ocean ecosystems, Shoal examines the fragile interdependence between marine life and the forces that disturb it. Visuals evolve in tandem with the sound, creating an immersive space where collective participation shapes the health or collapse of the underwater world.

My Contributions

Solo project - all aspects designed and implemented by Alicia Wu

  • Created browser-based animations replicating the movement of swarms and schools
  • Designed OSC data communication system to send animation data to Max for real-time audio processing
  • Implemented granular synthesis system for live audio manipulation
  • Developed interactive system using gyro data from audience phones to control audio parameters
  • Designed audience participation framework for collective control of live audio processing

Tools & Software

  • Max/MSP for real-time audio processing and granular synthesis
  • JavaScript for browser-based animation and swarm behavior algorithms
  • OSC (Open Sound Control) for communication between visual and audio systems
  • Mobile gyroscope sensors for audience interaction and parameter control
